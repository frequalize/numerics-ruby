# numerics.io API client for Ruby

A Ruby client for the (numerics.io)[https://numerics.io/] metrics API.

## Install

    $ sudo gem install numerics

## Summary

    require 'fz/numerics'

    client = Fz::Numerics.connect('project_access_key', 'project_secret_key')
    client.list # => []
